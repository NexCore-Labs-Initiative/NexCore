# سجل التغييرات

## v3.1.0 - ٢٦ يونيو ٢٠٢٦

### تحديث المبادرات والهوية

إصدار مركّز يجعل مبادرات NexCore Labs قابلة للاستكشاف عبر كتالوج ثنائي اللغة، ويضيف معاينات بطاقات هوية مصقولة لأعضاء الفريق.

#### ميزات جديدة

- إضافة صفحة **المبادرات** ليتمكن الزوار من استكشاف أعمال NexCore Labs المنشورة والنشطة والناشئة عبر كتالوج عام ثنائي اللغة.
- إضافة بطاقات عرض سريع للمبادرات مع حالات، وفلاتر تصنيف، ونقاط بارزة، وصور، وروابط إجراء رئيسية.
- إضافة **معاينات بطاقات هوية الفريق** من صور الأعضاء ليتمكن الزوار من عرض بطاقات هوية NexCore Labs مصقولة مع وصول تواصل عبر QR.

#### تحسينات

- إضافة رابط المبادرات عبر التنقل الرئيسي لتصبح الصفحة الجديدة متاحة من الصفحات الأساسية الإنجليزية والعربية.
- توسيع الروابط النظيفة، وإدخالات خريطة الموقع، وبيانات SEO، وتخزين Service Worker لتجربة المبادرات الجديدة.

### ملاحظات المطور

#### تغييرات تقنية

- إنشاء عارض المبادرات المشترك مع معالجة المحتوى المحلي، وفلاتر التصنيف، وعدّادات الملخص، وسلوك نافذة منبثقة قابل للوصول.
- إضافة دعم إنشاء المبادرات ومعاينتها ونشرها وحذفها من جهة الإدارة داخل لوحة التحكم الحالية.
- إضافة وحدة مخصصة لبطاقات هوية الفريق مع تحكم نافذة قابل للوصول، وحصر للتركيز، وأصول QR، ومشغلات صور محلية.

#### تغييرات قاعدة البيانات

- إضافة نموذج بيانات المبادرات للسجلات ثنائية اللغة، وحالات الظهور، والتصنيفات، والوسائط، والنقاط البارزة، وحقول دورة الحياة المدارة من الإدارة.

#### تحسينات داخلية

- إضافة اختبارات مركزة لصفحة المبادرات العامة، وتدفق إدارة المبادرات، ومعاينات بطاقات هوية الفريق.

## v3.0.0 - ٢٠ يونيو ٢٠٢٦

### اختراق الذكاء متعدد اللغات

إطلاق ثنائي اللغة كبير لـ NexCore Labs مع تعريب كامل، وهوية NexCore Intelligence، وتحسينات SEO دولية، ونقاط اشتراك جديدة، وأساس تقني أنظف للمنصة.

#### ميزات جديدة

- إطلاق تجربة **عربية كاملة** ضمن `/ar/` تشمل الصفحة الرئيسية، المركز، لوحة التحكم، صفحات المشاريع، الحساب، الصفحات القانونية، الأسعار، الإصدارات، ومحتوى الدعم.
- إعادة تقديم AI Chat باسم **NexCore Intelligence (NCI)** مع تجربة مخصصة في `intelligence.html` وتحديث التنقل في الصفحات الإنجليزية والعربية.
- إضافة تدفق **اشتراك النشرة البريدية** المدعوم بـ Supabase مع دعم إلغاء الاشتراك وتغطية عربية لصفحة الإلغاء.
- إضافة زر **حفظ الصفحة** في قسم التواصل مع سلوك مناسب للهواتف لتسهيل العودة إلى الموقع.
- إضافة زر **إنشاء بطاقة المشروع** في صفحات المشاريع ليتمكن الزوار من معاينة بطاقة مشروع مصممة وتنزيلها كملف PDF.

#### تحسينات

- تحسين SEO الدولي عبر `hreflang`، وبيانات وصفية محلية، ومعاينات اجتماعية عربية، وروابط نظيفة للصفحات الإنجليزية والعربية.
- تحسين اتجاه الواجهة العربية، والخطوط، ومحاذاة الشعار، وارتفاع الأسطر، ومبدل اللغة لتجربة ثنائية اللغة أكثر سلاسة.
- تحديث بطاقات المشاريع في المركز العربي بتصنيفات محلية وروابط تُبقي الزوار داخل تجربة المشروع العربية.
- تحديث واجهة التواصل بالرقم الرسمي لـ NexCore Labs وتحديث معلومات الفريق.
- تحديث قائمة فريق NexCore Labs بعد تقاعد محمد العاصمي من دور البيانات والتحليلات.
- تحديث هوية NexCore Labs البصرية بعلامة Core محسّنة، وأصول شعار نصي محدثة، وأيقونة تطبيق مخصصة لتجارب التثبيت.
- استبدال صفوف قائمة المستخدم المنفصلة بشريط `navUser` مدمج يضع الصورة والاسم بجانب أيقونات لوحة التحكم، والإدارة، والحساب، وتسجيل الخروج.
- تحسين القائمة المنسدلة باستبدال أيقونة المركز بنقاط Core ثابتة، ونقل رابط NexCore Intelligence بتصميم مميز بين زر الوصول إلى النواة وحقل البحث.

#### إصلاحات

- توسيع التخزين المسبق في Service Worker ودعم الروابط النظيفة لتصبح الصفحات المحلية وأصول الإصدارات أكثر موثوقية بعد النشر.
- إصلاح إرسال نموذج التواصل بحيث تنتقل الرسائل الناجحة إلى صفحات **الشكر** الخاصة بـ NexCore بدلاً من رابط النجاح المستضاف لدى Web3Forms.
- إزالة طبقة أتمتة الإصدارات التجريبية حتى يمكن إدارة محتوى الإصدارات يدوياً إلى حين إعادة تصميم التدفق لاحقاً.

### ملاحظات المطور

#### تغييرات تقنية

- إنشاء مجموعة الصفحات العربية ومواءمة التوجيه، والبيانات الوصفية، والتنقل، والواجهات المُولدة وقت التشغيل بين الإنجليزية والعربية.
- تحديث مسارات الواجهة والخلفية لـ NexCore Intelligence، بما في ذلك إعداد Gemini Flash ومسار intelligence الجديد.
- تحديث أسطح إصدار الموقع إلى `v3.0.0` بما يشمل بيانات الحزمة، وإصدار التشغيل، وروابط أصول الإصدارات، وإصدار ذاكرة Service Worker قبل النشر.
- تحديث ملف PWA manifest وقائمة التخزين المسبق في Service Worker لاستخدام مجموعة الأيقونات المحسّنة في التطبيقات المثبتة والعملاء المخزنين.
- إضافة وحدة محلية لتصدير بطاقة المشروع تبني المعاينة في المتصفح وتحمل مكتبات التقاط PDF عند طلب التنزيل فقط.
- تجميع تنقل المستخدم الموثق داخل مولّد واجهة المصادقة المشترك مع معرّفات إجراءات ثابتة وتغطية CSS في ملفي الأنماط الإنتاجي والمقروء.

#### تغييرات قاعدة البيانات

- إضافة معالجة اشتراكات النشرة البريدية عبر Supabase مع إبقاء تدفقات إلغاء الاشتراك متاحة للمستخدمين المحليين.

#### تحديثات API

- تحديث سلوك API الذكاء الاصطناعي حول مسار Gemini Flash مع إبقاء دعم استكشاف النماذج عبر مسار models.

#### تحسينات داخلية

- تفعيل Dependabot وتحديث إصدارات حزم npm لتعزيز الصيانة والأمان.
- إزالة تدفقات وسكربتات أتمتة الإصدارات لإبقاء قاعدة الكود أبسط إلى حين إعادة التفكير في عملية إصدار مستقبلية.
